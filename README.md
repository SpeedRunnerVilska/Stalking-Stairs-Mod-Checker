# Stalking Stairs Mod Checker

> A powerful in-game mod inspection overlay for **The Stalking Stairs**
> Built to detect, analyze, and visualize loaded mods in real time.

---

## Overview

**Stalking Stairs Mod Checker** is an advanced runtime mod detection and inspection tool designed for players, testers, and developers.

Instead of guessing what mods are loaded, this tool provides a **live interactive overlay** that displays:

* Loaded plugins
* Framework detection
* Risk analysis
* Metadata inspection
* Searchable mod lists
* Stylish UI themes

All directly inside the game.

---

## Features

### Mod Detection

* Automatically scans all loaded plugins
* Displays name, version, and GUID
* Real-time list updates

---

### Risk Analysis System

Each mod receives a **risk score** based on heuristics:

* Cheat/trainer keywords
* Suspicious identifiers
* Framework indicators
* Naming anomalies

Helps quickly identify potentially unsafe or gameplay-altering mods.

---

### Interactive Mod Inspector

* Click any mod to view detailed metadata
* GUID + version display
* Easy inspection for debugging and testing

---

### Smart Search

* Real-time filtering
* Searches mod name and GUID
* Instant results

---

### Stylish Overlay UI

* Multiple themes (Dark / Neon / Light)
* Smooth animated panel
* Draggable window
* Resizable interface
* Scrollable content
* Dimmed backdrop for focus

---

### Framework Detection

Automatically detects common modding environments and displays them inside the panel.

---

### Quality of Life

* Bottom HUD toggle button
* Scrollable mod list
* Animated open/close
* Clean credit display
* Lightweight runtime impact

---

## Installation

1. [Install **BepInEx** for the game](https://github.com/BepInEx/BepInEx/releases/download/v5.4.23.2/BepInEx_win_x64_5.4.23.2.zip)
2. Download the latest release DLL
3. Place DLL into:

```
BepInEx/plugins/
```

4. Launch the game
5. Press **Open Mod Panel** in-game
